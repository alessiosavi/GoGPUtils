# S3

The `aws/s3` package provides helpers for Amazon S3 operations, including object uploads, downloads, listing, and management.

## Features

- Object operations: Get, Put, Delete, Copy, Move
- Bucket operations: List objects with filtering
- Streaming uploads and downloads
- Automatic content type detection
- Parallel operations for batch processing

## Client Creation

```go
import "github.com/alessiosavi/GoGPUtils/aws/s3"

cfg, err := aws.LoadConfig(ctx, aws.WithRegion("us-west-2"))
if err != nil {
    return err
}

client, err := s3.NewClient(cfg)
if err != nil {
    return err
}
```

### Client Options

| Option                         | Description                     | Default |
| ------------------------------ | ------------------------------- | ------- |
| `WithUploaderConcurrency(n)`   | Concurrent upload parts         | 5       |
| `WithDownloaderConcurrency(n)` | Concurrent download parts       | 5       |
| `WithPartSize(size)`           | Part size for multipart uploads | 5 MB    |

```go
client, err := s3.NewClient(cfg,
    s3.WithUploaderConcurrency(10),
    s3.WithDownloaderConcurrency(10),
    s3.WithPartSize(10*1024*1024), // 10 MB
)
```

```go
type ObjectInfo struct {
```

```go
type ObjectMetadata struct {
```

```go
type DeleteError struct {
```

## Object Operations

### GetObject

Downloads an object from S3 and returns its contents.

```go
func (c *Client) GetObject(ctx context.Context, bucket, key string) ([]byte, error)
```

**Example:**

```go
data, err := client.GetObject(ctx, "my-bucket", "path/to/file.txt")
if errors.Is(err, s3.ErrObjectNotFound) {
    // Handle not found
}
```

### GetObjectReader

Returns a reader for streaming object content. The caller is responsible for closing the reader.

```go
func (c *Client) GetObjectReader(ctx context.Context, bucket, key string) (io.ReadCloser, *ObjectMetadata, error)
```

**Example:**

```go
reader, metadata, err := client.GetObjectReader(ctx, "my-bucket", "large-file.zip")
if err != nil {
    return err
}
defer reader.Close()

fmt.Printf("Size: %d bytes, Type: %s\n", metadata.ContentLength, metadata.ContentType)
io.Copy(dst, reader)
```

### PutObject

Uploads data to S3. Content type is automatically detected if not specified.

```go
func (c *Client) PutObject(ctx context.Context, bucket, key string, data []byte, opts ...PutOption) error
```

**Put Options:**

| Option                          | Description                |
| ------------------------------- | -------------------------- |
| `WithContentType(type)`         | Set content type           |
| `WithStorageClass(class)`       | Set storage class          |
| `WithServerSideEncryption(sse)` | Set server-side encryption |
| `WithMetadata(map)`             | Set custom metadata        |
| `WithCacheControl(header)`      | Set cache control header   |

**Example:**

```go
// Simple upload
err := client.PutObject(ctx, "my-bucket", "path/to/file.txt", []byte("content"))

// With options
err := client.PutObject(ctx, "my-bucket", "path/to/file.json", data,
    s3.WithContentType("application/json"),
    s3.WithStorageClass(types.StorageClassStandardIa),
    s3.WithServerSideEncryption(types.ServerSideEncryptionAes256),
    s3.WithMetadata(map[string]string{
        "author": "john",
        "version": "1.0",
    }),
)
```

### PutObjectReader

Uploads data from a reader to S3.

```go
func (c *Client) PutObjectReader(ctx context.Context, bucket, key string, reader io.Reader, opts ...PutOption) error
```

**Example:**

```go
file, err := os.Open("large-file.zip")
if err != nil {
    return err
}
defer file.Close()

err = client.PutObjectReader(ctx, "my-bucket", "uploads/file.zip", file,
    s3.WithContentType("application/zip"),
)
```

### DeleteObject

Deletes an object from S3.

```go
func (c *Client) DeleteObject(ctx context.Context, bucket, key string) error
```

**Example:**

```go
err := client.DeleteObject(ctx, "my-bucket", "path/to/file.txt")
```

### DeleteObjects

Deletes multiple objects from S3. Maximum of 1000 objects per call (AWS limit).

```go
func (c *Client) DeleteObjects(ctx context.Context, bucket string, keys []string) ([]string, []DeleteError)
```

**Example:**

```go
keys := []string{"file1.txt", "file2.txt", "file3.txt"}
deleted, errors := client.DeleteObjects(ctx, "my-bucket", keys)

for _, err := range errors {
    log.Printf("Failed to delete %s: %s", err.Key, err.Error)
}
```

### CopyObject

Copies an object within or between buckets.

```go
func (c *Client) CopyObject(ctx context.Context, srcBucket, srcKey, dstBucket, dstKey string) error
```

**Example:**

```go
// Copy within same bucket
err := client.CopyObject(ctx, "my-bucket", "source.txt", "my-bucket", "dest.txt")

// Copy to different bucket
err := client.CopyObject(ctx, "source-bucket", "file.txt", "dest-bucket", "file.txt")
```

### MoveObject

Moves an object by copying then deleting the source.

```go
func (c *Client) MoveObject(ctx context.Context, srcBucket, srcKey, dstBucket, dstKey string) error
```

**Example:**

```go
err := client.MoveObject(ctx, "my-bucket", "old-path/file.txt", "my-bucket", "new-path/file.txt")
```

### ObjectExists

Checks if an object exists in the bucket.

```go
func (c *Client) ObjectExists(ctx context.Context, bucket, key string) (bool, error)
```

**Example:**

```go
exists, err := client.ObjectExists(ctx, "my-bucket", "path/to/file.txt")

    fmt.Println("Object exists")

```

Retrieves object metadata without downloading the content.


## Bucket Operations

### ListObjects

Returns all objects in a bucket matching the options. Results are sorted by LastModified ascending.

```go
func (c *Client) ListObjects(ctx context.Context, bucket string, opts ...ListOption) ([]ObjectInfo, error)
```

**List Options:**

| Option                  | Description                         |
| ----------------------- | ----------------------------------- |
| `WithPrefix(prefix)`    | Filter by key prefix                |
| `WithDelimiter(delim)`  | Set delimiter for grouping          |
| `WithMaxKeys(n)`        | Limit number of objects             |
| `WithStartAfter(key)`   | Start listing after key             |
| `WithModifiedAfter(t)`  | Filter objects modified after time  |
| `WithModifiedBefore(t)` | Filter objects modified before time |

**Example:**

```go
// List all objects
objects, err := client.ListObjects(ctx, "my-bucket")

// List with prefix
objects, err := client.ListObjects(ctx, "my-bucket", s3.WithPrefix("logs/"))

// List with time filter
since := time.Now().Add(-24 * time.Hour)
objects, err := client.ListObjects(ctx, "my-bucket", s3.WithModifiedAfter(since))

for _, obj := range objects {
    fmt.Printf("%s (%d bytes)\n", obj.Key, obj.Size)
}
```

### ListObjectKeys

Returns just the keys of objects in a bucket. More efficient than ListObjects when you only need keys.

```go
func (c *Client) ListObjectKeys(ctx context.Context, bucket string, opts ...ListOption) ([]string, error)
```

**Example:**

```go
keys, err := client.ListObjectKeys(ctx, "my-bucket", s3.WithPrefix("data/"))
```

### ListObjectsCallback

Iterates through objects and calls a callback for each. Useful for processing large numbers of objects without loading all into memory.

```go
func (c *Client) ListObjectsCallback(ctx context.Context, bucket string, callback func(ObjectInfo) error, opts ...ListOption) error
```

**Example:**

```go
err := client.ListObjectsCallback(ctx, "my-bucket", func(obj s3.ObjectInfo) error {
    fmt.Printf("Found: %s (%d bytes)\n", obj.Key, obj.Size)
    return nil
}, s3.WithPrefix("logs/"))
```

### CountObjects

Returns the count of objects matching the options.

```go
func (c *Client) CountObjects(ctx context.Context, bucket string, opts ...ListOption) (int64, error)
```

**Example:**

```go
count, err := client.CountObjects(ctx, "my-bucket", s3.WithPrefix("logs/"))
fmt.Printf("Total objects: %d\n", count)
```

### TotalSize

Returns the total size in bytes of objects matching the options.

```go
func (c *Client) TotalSize(ctx context.Context, bucket string, opts ...ListOption) (int64, error)
```

**Example:**

```go
size, err := client.TotalSize(ctx, "my-bucket", s3.WithPrefix("logs/"))
fmt.Printf("Total size: %.2f MB\n", float64(size)/1024/1024)
```

## Utility Functions

### ParseS3Path

Parses an S3 URI (`s3://bucket/key`) into bucket and key components.

```go
func ParseS3Path(s3Path string) (bucket, key string, err error)
```

**Example:**

```go
bucket, key, err := s3.ParseS3Path("s3://my-bucket/path/to/file.txt")
// bucket = "my-bucket", key = "path/to/file.txt"
```

### FormatS3Path

Formats a bucket and key into an S3 URI.

```go
func FormatS3Path(bucket, key string) string
```

**Example:**

```go
path := s3.FormatS3Path("my-bucket", "path/to/file.txt")
// path = "s3://my-bucket/path/to/file.txt"
```

## Error Handling

### Sentinel Errors

| Error               | Description                  |
| ------------------- | ---------------------------- |
| `ErrObjectNotFound` | Object does not exist        |
| `ErrBucketNotFound` | Bucket does not exist        |
| `ErrAccessDenied`   | Access to resource is denied |

### Error Handling Example

```go
data, err := client.GetObject(ctx, "bucket", "key")
if err != nil {
    if errors.Is(err, s3.ErrObjectNotFound) {
        // Handle not found - create default or return 404
        return nil
    }

    var opErr *aws.OperationError
    if errors.As(err, &opErr) {
        log.Printf("S3 operation failed: %s", opErr.Operation)
    }

    return err
}
```

## Testing

```go
type mockS3API struct {
    s3.API
    getObjectData []byte
}

func (m *mockS3API) GetObject(ctx context.Context, params *s3sdk.GetObjectInput, optFns ...func(*s3sdk.Options)) (*s3sdk.GetObjectOutput, error) {
    return &s3sdk.GetObjectOutput{
        Body: io.NopCloser(bytes.NewReader(m.getObjectData)),
    }, nil
}

func TestMyFunction(t *testing.T) {
    mock := &mockS3API{getObjectData: []byte("test content")}
    client := s3.NewClientWithAPI(mock, nil, nil)

    data, err := client.GetObject(context.Background(), "bucket", "key")
    if err != nil {
        t.Fatal(err)
    }

    if string(data) != "test content" {
        t.Errorf("unexpected data: %s", data)
    }
}
```
